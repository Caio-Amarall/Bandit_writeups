# Bandit level 17 → 18

## Objetivo
Temos dois arquivos para a senha, existe apenas uma única linha
de diferença entre os dois arquivos.

## Solução
Devemos usar o comando diff para que encontremos a 
única linha de diferença.

# Comandos usados
```bash
ls
diff passwords.old passwords.new
```
## Aprendizado

# O comando diff
Compara dois arquivos e mostra as diferenças

## Conclusão
Após dada a diferença, encontramos a senha
para o próximo nível.
