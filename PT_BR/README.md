# 🛡️ OverTheWire Bandit Writeups

Este repositório contém meus write-ups completos de todos os níveis do **OverTheWire Bandit**, um wargame focado no aprendizado prático de Linux e fundamentos de segurança.

---

## 📌 Sobre o desafio

O **Bandit** é um dos desafios mais recomendados para iniciantes em cibersegurança.
Ele ensina, na prática, como navegar em sistemas Linux, manipular arquivos e entender conceitos fundamentais usados em segurança ofensiva.

---

## 🎯 Objetivo deste repositório

* Documentar minha jornada completando todos os níveis
* Consolidar meu aprendizado em Linux e segurança
* Criar um portfólio prático para cibersegurança

---

## 🧠 O que eu aprendi

Durante os desafios, desenvolvi habilidades como:

* Navegação em sistemas Linux (`cd`, `ls`, `file`)
* Leitura e manipulação de arquivos (`cat`, `less`, `strings`)
* Uso de comandos avançados (`grep`, `find`, `sort`, `uniq`)
* Permissões e propriedade de arquivos (`chmod`, `chown`)
* Redirecionamento e pipes (`>`, `>>`, `|`)
* Noções básicas de segurança e exploração
