# Bandit level 7 → 8

## Objetivo
Precisamos ler o arquivo data.txt e encontrar uma palavra especifica
"millionth" que estará junto ao código. O arquivo de texto tem
milhares de palavras.

## Solução
Usaremos o comando "grep" para encontrar a palavra em meio a milhares
de outras palavras.

# Comandos usados
```bash
ls
grep "millionth" data.txt
```

## Aprendizado

# Comando Grep
O comando "grep" tem a função de procurar por uma palavra especifica
dentro de um arquivo de texto. 

## Conclusão
Ao encontrarmos a palavra mencionada, obtemos o código e usamos para
a conexão da próxima fase. O código é dfwvzFQi4mU0wfNbFOe9RoWskMLg7eEc.
