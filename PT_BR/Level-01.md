# Bandit level 0 → 1

## Objetivo
Precisamos encontrar uma pasta chamada readme que esta localizada no diretório home.
Devo usar a senha encontrada na pasta para conectar no servidor bandit1.

## Solução
Temos as informações de onde está localizado a pasta e onde devemos usar a senha.

# Comandos usados
```bash
ls
cat readme
```
## Aprendizado

# Ls
O comando ls do linux mostra quais arquivos ou pastas tem no diretório que estamos.
Devemos lembrar, apenas o ls não mostra pastas ou arquivos que estão ocultos. Devemos
usar a opção -alps para que apareça todo tipo de arquivos, quais as permissões de
cada arquivo, a quantidade de pastas/arquivos e outras informações.

# Cat
O comando cat é um comando do linux para ler um arquivo de texto.

## Conclusão
Depois de ler o arquivo readme que encontramos, nós temos a senha.
Vamos para o próximo nivel.
